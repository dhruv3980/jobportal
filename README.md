# Job Board

## 📌 Overview

This project is a **Job Board** built using **Vanilla HTML, CSS, and JavaScript**.
It displays job listings in a card layout and allows users to filter jobs based on location, salary, and job title.

---

## 🚀 Features

* Display job listings in a responsive card layout
* Filter jobs by **location (dropdown)**
* Filter jobs by **minimum salary (input field)**
* Search jobs by **title (real-time search)**
* Show **"No results found"** when no jobs match

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🧪 Functionality

### 🔹 2. Location Filter

### 🔹 3. Salary Filter

### 🔹 4. Search by Title

### 🔹 5. No Results Handling
